Универсальный AI оркестратор

Мозг моих проектов, использующий лучшие AI решения на рынке как для быстрых MVP так и для больших и сложных проектов, координирует работу ИИ агентов как команды, где у каждого своя роль:
Codex - архитектура и ревью, 

Claude code - план и разработка, 

Antigravity - визуал тесты, 

Оркестратор самостоятельно маршрутизирует задачи к наиболее подходящим ИИ и конкретным моделям, экономит контекст там где модель избыточна, контролирует качество через обязательное использование **hard gates (plugins, skills, test, architecture и др.) и engineering playbook (review handoff, test report** , k**nown risks, skill trace и др.)** **:**

Skills - библиотека из проверенных навыков для ИИ-ассистентов от Anthropic, OpenAI, Google, Microsoft, Supabase и Vercel Labs и др., покрывающих весь цикл разработки. Позволяет быстро собрать команду экспертов по архитектуре, деву, данным, безопасности, тестированию и автоматизации без написания всего с нуля.

Plugins — встроенные расширения Claude Code, которые подключают внешние инструменты: TypeScript-анализ, браузерное тестирование, code review через GitHub, аудит безопасности и др.

**Engineering playbook (как часть архитектуры):**

Набор форматов и правил для ключевых событий цикла разработки: 

Исключает потерю контекста между участниками и сессиями — каждый знает, что и в каком виде передавать, без согласования в чате.

- **Review Handoff** — структурированная передача задач в ревью: что сделано, какие файлы изменены, какие риски остаются.
- **Test Report** — зафиксированные результаты проверок: какие команды запускались и с каким результатом. Без неподтверждённых заявлений.
- **Known Risks** — централизованный список рисков до мерджа.
- **Skill Trace** — фиксация применённой экспертизы: какие навыки использованы и почему.
- **Детерминизм** — жёсткие правила для нестабильных зон (таймзоны, сортировки, кэш, deny-тесты) для устранения случайных сбоев.

Компетенции: Визуализация, автоматизация workflows, CI/CD, самодокументируемые инструменты, AI логика и архитектура и др.
